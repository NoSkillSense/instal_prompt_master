# Instalator Prompt Master

Skrypt instalacyjny dla aplikacji Prompt Master (Ubuntu).

## Uruchomienie

```bash
curl -fsSL https://raw.githubusercontent.com/NoSkillSense/instal_prompt_master/main/install.sh | bash
```

Lub:

```bash
git clone https://github.com/NoSkillSense/instal_prompt_master.git
cd instal_prompt_master
./install.sh
```
